Встановлення:
1. Відкрити командний рядок (у пошуку ввести cmd).
2. Перевірити чи є Python (у командному рядку виконати python --version). Якщо немає, то встановити Python.
3. Оновити менеджер пакетів pip (виконати pip install --upgrade pip)
4. У командному рядку перейти до каталогу IP-decoding-docx
5. Виконати pip install -r requirements.txt
___________________________________________________________

Запуск:
1. Відкрити файл const\constants.py у режимі редагування, вписати у змінну IN_FILE назву вхідного файлу, наприклад IN_FILE = 'example.docx'
2. У командному рядку виконати python main.py або запустити run_script.bat
____________________________________________________________

Для виконання дешифрування IP в кількості більш ніж 1000 за день або більш ніж 30000 на місяць необхідно на https://ipinfo.io/ створити обліковий запис та у змінну ITERATOR_TOKENS (файл const\constants.py) вставити сгенерований токен, наприклад ITERATOR_TOKENS = iter(('', 'qwerty...', '', '', ''))
Один обліковий запис = 50000 запитів на місяць

LIMIT_DAY = 5 це кількість днів, після яких вважати, що розшифровані IP вже застаріли після збереження їх у базу ip_resolved_list.db (розшифровані IP зберігаються у ip_resolved_list.db для підвищення швидкості виконання дешифрування та зменшення виконнання запитів до https://ipinfo.io/)
